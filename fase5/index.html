Introducción

Introduction

Este entorno se desarrolló con el propósito de dar conocer la estructura de HTML Elementos y etiquetas que nos apoyaran en el momento de definir la capa basica de una web.

El lenguaje HTML (hypertext markup language) es el lenguaje con el que se definen páginas Web.
Permite describir el contenido de una página, incluyendo texto y otros elementos (imágenes, vídeo, otros documentos, etc.).

HTML es la tecnología fundamental de la World Wide Web y ha ido evolucionando hasta nuestros días.

This environment was developed with the purpose of making HTML's structure of common knowledge, including Elements and tags that will support us at the moment of defining a web's basic layer.

HTML languaje(hypertext markup language) is the languaje with which web pages are defined.
Allows to describe the contents of a web page, incliding text and other elements (images, video, other documents, etc).

HTML is the fundamental technology in the World Wide Web and it has been evolving up to the date.
